Build your web page...

  1. Create a folder on your desktop - label it "home_page".
  2. Create two empty files with NotePad++ (or similar editor - NOT MS Word!!).
  3. Save them as index.html and style.css.
  4. Copy this text into index.html.
  5. Copy this text into style.css.
  6. Save your files.
  7. Right click on the index.html file and select the “Open with” option and specify your editor.
  8. You can change anything in either file that is in red in the Google Doc without having a problem displaying your web page. Color names can be found here
  9. To view your work:
    Open your browser.
    Click on File->Open File.
    Navigate to your index.html file and select it.
